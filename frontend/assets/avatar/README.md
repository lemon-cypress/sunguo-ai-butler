# Sunguo avatar model slot

Put the first VRoid export here:

- `sunguo.vrm`

Recommended VRoid export settings for the first prototype:

- Format: VRM
- Pose: neutral / standard standing pose
- Expression: neutral or soft smile
- Outfit: daily gentle butler / home assistant style
- Keep the file name exactly `sunguo.vrm`

The web dashboard will automatically try this path:

`frontend/assets/avatar/sunguo.vrm`

If the file is missing, the dashboard keeps using the built-in low-poly placeholder, so the page will not break.
