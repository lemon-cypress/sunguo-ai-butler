# 早报质量升级 v2

## 目标

让松果从“罗列数据”升级成“先给判断，再给证据和核验点”。

当前 v2 增加了：

- 今日总览
- 三件最重要的事
- 为什么重要
- 影响对象
- 证据
- 可信度
- 后续核验点

## 输出位置

结构化数据：

```text
morning_brief_input.json -> brief_analysis
output_bundle.json -> screen_cards
```

文本早报：

```text
morning_brief.txt 顶部的“松果解读”
```

网页：

```text
http://localhost:8765/frontend/
```

页面会优先展示：

- 松果解读
- 1/2/3 三件重要事
- 原有天气、市场、主题、公司、日程卡片

## 可信度规则

- `高`：来自本地日程、明确结构化输入、或稳定事实
- `中`：来自行情数据、天气数据等结构化接口
- `线索`：来自 RSS、Google News 标题、第三方新闻标题
- `待确认`：数据不足或来源不明确

## 后续扩展

下一步可以在这个基础上做：

- 追问能力：围绕 `brief_analysis.top_stories` 继续问为什么
- 本地记忆：记住你关注的国家、行业、公司和提醒偏好
- 语音质量：把 `top_stories` 作为 TTS 开场重点
- 数字人动作：重要事项用更聚焦的表情和手势
