# 早报解读层

## 为什么需要解读层

直接把市场指数和新闻标题交给模型，容易输出成“标题列表”。松果应该更像管家：先帮用户判断哪些值得关注，再温柔地讲出来。

## 当前解读内容

新增 `insights` 字段，包括：

- `market`：市场情绪、明显变化指数、区域概览、后续关注点。
- `news`：新闻主题线索、不同栏目头条、后续关注点。
- `themes`：商品/主题价格上涨线索、可能影响行业、机会观察。
- `priorities`：今日行动建议。

## 查看方式

```powershell
cd E:\松果\ai-butler
python .\backend\app\morning_brief_demo.py --show-json
```

查看输出中的：

```json

  "market": {},
  "news": {},
  "priorities": []
}
```

## 注意

- 解读层不是投资建议。
- 新闻标题只是线索，仍需要核验来源和原文。
- 后续可以继续增加行业归因、公司公告、商品价格、政策事件等专门解读器。
