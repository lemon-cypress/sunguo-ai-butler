# 第 1 周行动清单

## 本周目标

把“松果”从想法变成一个清楚的项目定义，并为第 2 周写代码做好准备。

## 已确定信息

- 项目名称：松果
- AI 形象：温柔、阳光、聪明的 18 岁女生
- 第一版核心问题：
  1. 每天汇总前一天全球关键事件，包括经济、政治、股市等。
  2. 讲当天的天气和着装建议。
  3. 接入 Outlook，说清楚今天有哪些需要做的事情。

## 你当下需要做的事

### 1. 安装基础工具

本周只需要安装这些：

- VS Code
- Python 3.11 或 Python 3.12
- Git
- GitHub Desktop

### 2. 准备账号和权限

需要准备：

- OpenAI API Key，后面用于 Agent。
- Microsoft 账号，后面用于 Outlook 日历接入。
- 一个 GitHub 账号，用于保存项目。

### 3. 写 10 条真实早晨需求

请写下你希望松果早上告诉你的 10 类信息，例如：

- 昨天全球最重要的 5 件事。
- 今天有没有影响股市的大事。
- 今天上海天气如何。
- 今天穿什么更合适。
- 上午有哪些会议。
- 哪个会议最重要。
- 有没有需要提前准备的材料。
- 今天必须完成的 3 件事。
- 哪些事情可以推迟。
- 晚上是否需要复盘。

### 4. 确认默认城市

请确认第一版天气使用哪个城市。默认可以先用：

- 上海

### 5. 确认 Outlook 范围

第一版 Outlook 只读，不修改日历。

需要确认读取范围：

- 今天日程。
- 明天日程。
- 本周待办。
