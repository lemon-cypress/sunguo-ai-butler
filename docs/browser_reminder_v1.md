# 浏览器提醒 v1

目标：先让松果在网页打开时真的能提醒，而不是只生成提醒计划。

## 当前能力

- `frontend/app.js` 会读取 `reminder_plan.items`。
- 页面会动态显示“主动提醒”运行面板。
- 点击“启用提醒”后，浏览器会请求系统通知权限。
- 页面保持打开时，会按今天剩余提醒时间设置计时器。
- 提醒触发时会：
  - 更新页面内提醒状态；
  - 使用浏览器语音朗读提醒；
  - 在权限允许时发送浏览器系统通知。
- 点击“测试提醒”可以立刻验证提醒声音和通知权限。

## 注意事项

- 这一版不是后台常驻服务，网页关闭后不会继续提醒。
- 浏览器通知权限需要用户手动允许。
- 后续可升级为 Windows 常驻进程、托盘程序或手机推送。
